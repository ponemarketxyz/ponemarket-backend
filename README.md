# PoneMarket Backend API

Node.js Express API for PoneMarket paper trading platform.

## Deploy to Railway
1. Upload this folder to GitHub
2. Connect repo to Railway
3. Set PORT environment variable (Railway sets this automatically)

## Endpoints
- POST /v1/agents/register
- GET  /v1/markets
- POST /v1/orders
- GET  /v1/portfolio
- GET  /v1/leaderboard
